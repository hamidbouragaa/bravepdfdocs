BravePDF - AI-Powered PDF Tool Suite

BravePDF is a comprehensive, AI-powered web application for all your PDF needs. Built with a modern tech stack, it provides a suite of powerful tools ranging from document conversion and optimization to intelligent analysis and interaction. This source code provides a ready-to-deploy, subscription-based SaaS application that you can sell to your own customers.

Features

  • AI-Powered Tools: Summarize, translate, chat with, analyze, and generate PDFs using AI. Includes an OCR scanner and semantic search.
  • Convert & Create: Convert PDFs to and from formats like Word, PowerPoint, Image, HTML, Text, CSV, XML, and EPUB.
  • Organize & Optimize: Merge, split, compress, rotate, reorder pages, manage bookmarks, and compare PDF versions.
  • Edit & Sign: eSign documents, crop pages, remove backgrounds, create and fill forms, and redact sensitive information.
  • Security: Protect with passwords, unlock secured files, add watermarks, and scan for viruses.
  • Business & Productivity: Generate professional invoices, reports, certificates, and QR codes.
  • SaaS & Monetization:
    • User authentication (Login/Signup).
    • Multi-tier subscription plans (Free, Pro, Pro+).
    • Seamless checkout process powered by Stripe.
    • Feature gating based on subscription plans.
    • User dashboard to manage subscriptions and usage.
    • Superadmin panel for site management.

Tech Stack

  • Framework: Next.js (App Router)
  • Styling: Tailwind CSS & ShadCN UI
  • AI Backend: Google AI & Genkit
  • Database & Auth: Firebase (Authentication & Firestore)
  • Payments: Stripe
  • Deployment: Vercel (Recommended)

Prerequisites

  1. Node.js: v18.x or later
  2. Firebase Account: To manage users, data, and service credentials
  3. Google AI (Gemini) API Key: To power the AI features
  4. Stripe Account: To process subscription payments
  5. Vercel Account: For easy deployment and hosting

Step-by-Step Setup Instructions

1. Initial Project Setup

  1. Download and extract the source code folder.
  2. Open your terminal, navigate into the project directory, and install dependencies: 
    npm install
  3. Rename .env.example to .env.local: 
    mv .env.example .env.local
  4. Fill in your .env.local file with your actual values.

2. Firebase Configuration

  1. Create a Firebase Project via the Firebase Console.
  2. Create a Web App, copy firebaseConfig values to .env.local.
  3. Enable Firestore (Production mode) and Email/Password Authentication.
  4. Create a Service Account, copy project_id, client_email, and private_key to .env.local (escape newlines with \n).

3. Stripe Configuration

  1. Create "Pro" and "Pro+" products with recurring monthly prices in Stripe dashboard. Update Price IDs in src/app/pricing/page.tsx.
  2. Copy Publishable and Secret keys into .env.local.
  3. Add a webhook endpoint (https://<YOUR_VERCEL_URL>/api/stripe-webhook) with events: checkout.session.completed, invoice.payment_succeeded. Paste signing secret into STRIPE_WEBHOOK_SECRET.

4. Google AI (Gemini) API Key

  1. Get an API key from Google AI Studio and place it in GEMINI_API_KEY in .env.local.

5. Superadmin Access

  1. Sign up for an account in the app.
  2. Find your UID in Firebase Authentication.
  3. Update your Firestore user documentâ€™s role to "superadmin".
  4. Log out and log back in to see the admin panel.

Deployment to Vercel

  1. Push code to a GitHub repository.
  2. Import repo into Vercel.
  3. Add environment variables from .env.local into Vercel settings.
  4. Deploy the project.
  5. Update Stripe webhook URL to production URL.

Local Development

npm run dev

Runs the app at http://localhost:9002.

Congratulations! Your AI-powered PDF SaaS is now fully configured and ready to be marketed.
